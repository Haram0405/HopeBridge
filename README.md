🌉 HopeBridge Donation Management System

HopeBridge is a frontend-based Donation Management System developed using HTML, CSS, Bootstrap, and JavaScript. The project is designed to help manage donation activities through a clean and user-friendly interface.

✨ Features

✅ Responsive Sidebar Navigation 

✅ Interactive Dashboard

✅ Donation Causes Section 

✅ Category Filter Buttons

✅ View & Donate Modals 

✅ LocalStorage Integration

✅ Donation Givers Management 

✅ Reports & Statistics Section

✅ Edit & Delete Donation Functionality 

✅ Needy Contacts Page

✅ About & Contact Section

🛠 Technologies Used
HTML5
CSS3
Bootstrap 5
JavaScript
LocalStorage

🚀 Project Purpose

The main goal of this project was to practice frontend development concepts by building a real-world style donation management system with interactive functionality and a professional UI.
